What is the Document Object Model (DOM)?

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of a webpage as a hierarchical tree of elements, allowing developers to access, modify, and update HTML or XML content dynamically using JavaScript or other programming languages.

How do you select an element from the DOM?

To select an element from the DOM, use JavaScript methods like document.getElementById('id'), document.querySelector('.class'), or document.querySelectorAll('tag'). These allow you to retrieve elements by their ID, class, or tag for further manipulation or interaction.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element, allowing it to manage events for multiple child elements. This improves performance and simplifies event handling.

How do you manipulate an element's attributes and styles using the DOM?

Select an element using document.querySelector(), then use setAttribute() to change attributes and modify styles with element.style.propertyName = 'value' for dynamic updates in the DOM.
